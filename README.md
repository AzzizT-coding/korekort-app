# Kørekort Pris Sammenligning 🏍️

En interaktiv web-app til at sammenligne gennemsnitspriser på kørekort i forskellige danske byer.

## Funktioner

✅ **Prissammenligning** - Se og sammenlign kørekort priser i 12+ danske byer  
✅ **Søgefunktion** - Find byer hurtigt  
✅ **Sortering** - Sorter efter by navn eller pris (høj/lav)  
✅ **Favoritter** - Gem dine favorit-byer (gemmes i browser)  
✅ **Interaktive diagrammer** - Visualiser priser med søjlediagrammer  
✅ **Responsivt design** - Fungerer på mobil, tablet og desktop  

## Byer inkluderet

- København (Hovedstaden)
- Aarhus (Midtjylland)
- Odense (Fyn)
- Aalborg (Nordjylland)
- Og mange flere...

## Installation & Brug

1. Åbn `index.html` i din browser
2. Brug søgekassen til at finde byer
3. Klik på ❤️ knapperne for at gemme favoritter
4. Sorter efter pris eller by navn
5. Se priserne visualiseret i diagrammet

## Teknologi

- **HTML5** - Struktur
- **CSS3** - Styling og responsive design
- **JavaScript** - Funktionalitet og interaktivitet
- **Chart.js** - Diagramvisualisering
- **LocalStorage** - Gemme favoritter

## Filer

- `index.html` - Hovedside
- `styles.css` - Styling
- `app.js` - Applikationslogik
- `data.js` - Bydata og priser

## Noter

Priserne er gennemsnitspriser og kan variere afhængigt af kørekortskole og kursustype.
